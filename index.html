COBERTURA VEGETAL

Índice vegetación de
diferencia normalizada
NDVI

NDVI Negativo indica ausencia de vegetación
NDVI < 0.4 señala vegetación escasa
0.4 < NDVI < 0.6 indica vegetación moderada
NDVI > 0.6 indica alta cobertura vegetal en más del 60% del área

ESTRUCTURA VEGETAL

Índice vegetación
de suelo modificado
MSAVI


MSAVI Negativo indica ausencia de vegetación
0 < MSAVI < 0.2 señala vegetación herbácea escasa
0.2 < MSAVI < 0.4 es presencia de pastizales y arbustos
0.4 < MSAVI < 1 indica una estructura vegetal leñosa

CONECTIVIDAD INTEGRAL

X < 0.25 indica vegetación en parches aislados con menos del 25% de cobertura vegetal leñosa
0.25 < X < 0.5 señala vegetación en "parches" cubriendo el 50% del área
0.5 < X < 0.75 es vegetación arbórea y arbustiva cubriendo hasta el 75% del área
0.75 < X es un corredor denso en el ecosistema con vegetación en más del 75% del predio
X < 2 refiere a una mala conectividad de espacios entre senderos
2 < X < 4 indica una conectividad poco eficaz
4 < X < 6 es un corredor aclarado con buena conectividad entre áreas
6 < X indica una excelente conectividad entre áreas
